# 📊 Sales Analysis Dashboard (Streamlit App)

An interactive **Sales Data Analysis Dashboard** built using Python and Streamlit.  
This project analyzes sales data from a CSV file and provides **real-time business insights, KPIs, and visual analytics**.

---

## 🚀 Live Demo

👉 https://sales-analysis-dashboard-myrfv7cjzj9tu5l7btjdio.streamlit.app/

---

## 📌 Project Overview

This project is an end-to-end sales analytics solution that helps in understanding business performance through:

- Data loading and preprocessing
- Revenue calculation
- KPI generation
- Product-wise analysis
- Region-wise performance analysis
- Monthly sales trend visualization
- Interactive dashboard using Streamlit

---

## 🛠 Technologies Used

- Python 🐍
- Streamlit 📊 (Web App Framework)
- Pandas 📊 (Data Analysis)
- Matplotlib 📈 (Data Visualization)
